关于 IconApron
实用、方便、快速使用在多平台的图标库

IconApron 图标库是一个配合 FunPlus UI 组件库各种使用场景产生的开源图标库，从而保证了图标在各个场景下的一致性。

使用统一风格的图标可以使得产品保持统一的风格和认知。IconApron 主要保持了 FunPlus UI 的设计风格，拥有一致、统一的颜色、尺寸等方面的设计。让使用 FunPlus UI 的项目可以轻松的找到合适的图标

Github: https://github.com/offontime/iconapron
